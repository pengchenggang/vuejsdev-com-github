Step. 4: 打包发布 ​

全局安装 vsce

js
npm install -g vsce
1

Step. 5: 本地打包 ​

打包插件，让修改README.md 改成了这个插件的说明文档后，就可以打包

js
vsce package
1

Step. 6: 上传vscode插件库 ​

将自己的插件传到公网上，通过搜索AutoNumber就可以找到我的插件

  • publishing-extension
  • create-organization
  • 发布到插件市场了，搜索 autoNumber 即可

Down. 7: 下载插件 ​

点击插件 右上角... 选择 从VSIX 安装

  • 第一个开发vscode插件 还有点小兴奋
  • autonumber-for-vscode-0.0.1.vsix 下载
